# special_python_methods.1b
This repository contains a Jupyter Notebook demonstrating some python special methods(also known as dunder methods or magic methods)

### Covered Methods include:
1. __init__- object Constructor
2. __str__ & __repr__- string representation
3. __bool__- boolean truthiness
4. __add__  & __sub__- operator
5. __contains__ - membership testing
6. __del__- object destructor
